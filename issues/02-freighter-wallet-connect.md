# Add Freighter Wallet Connect

- Complexity: Medium
- Points: 150
- Labels: `stellar-wave`, `frontend`, `wallet`

## Problem
The UI uses manual sender text input and does not connect to a real wallet.

## Scope
- Add Freighter wallet connect/disconnect buttons.
- Autofill sender from connected wallet.
- Block stream creation if wallet is not connected.

## Acceptance Criteria
- User can connect Freighter from UI.
- Sender account is loaded from wallet.
- Stream creation flow validates wallet connection.
